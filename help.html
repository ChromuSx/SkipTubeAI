SkipTube AI - Help Guide

AI-powered YouTube content skipper

rocket_launch Getting Started

Follow these simple steps to start using SkipTube AI:

1 Get Your API Key

You need a Claude AI API key from Anthropic:

  1. Go to console.anthropic.com
  2. Create an account or log in
  3. Navigate to API Keys section
  4. Create a new API key and copy it

Note: Your API key is stored locally in your browser and never shared with anyone except Anthropic's API for video analysis.

2 Configure the Extension

  1. Click the SkipTube AI extension icon in your browser
  2. Paste your API key in the settings
  3. Choose which content types you want to skip (Sponsors, Intros, Outros, etc.)
  4. Save your settings

3 Watch YouTube Videos

That's it! The extension will automatically analyze videos and skip unwanted segments when you watch YouTube.

settings Features & Settings

Content Types

Choose which types of content to skip:

  • Sponsors: Product/service sponsorships and ads
  • Intros: Video introductions and opening sequences
  • Outros: End screens and closing sequences
  • Self-Promotion: Channel self-promotion and merchandise
  • Donations: Super chat acknowledgments and donation mentions

Advanced Settings

  • Skip Buffer: Time in seconds before segment to start skipping (default: 0.5s)
  • Confidence Threshold: AI confidence level required to skip (0-1, default: 0.85)
  • Channel Whitelist: Add channels to exclude from analysis
  • Enable Preview: Show notification before skipping

psychology How It Works

  1. When you watch a YouTube video, the extension extracts the video transcript
  2. The transcript is sent to Claude AI for analysis
  3. AI identifies sponsorships, intros, outros, and other segments
  4. Results are cached locally to avoid re-analyzing the same video
  5. The extension automatically skips identified segments based on your preferences

Important: This extension only works with videos that have subtitles/transcripts available.

help_outline Frequently Asked Questions

Q: Why isn't the extension working on a video?
A: The video must have subtitles/transcripts available. The extension cannot analyze videos without transcripts.
Q: Is my data private?
A: Yes! All settings and cache are stored locally on your device. Only video transcripts are sent to Anthropic's API for analysis. See our Privacy Policy for details.
Q: Does this cost money?
A: You need to pay for Claude AI API usage through Anthropic. The cost is very low (typically fractions of a cent per video). The extension itself is free and open source.
Q: Can I whitelist specific channels?
A: Yes! Go to Advanced Settings and add channel handles or IDs to the whitelist. Videos from those channels won't be analyzed.
Q: How do I clear the cache?
A: Click the extension icon, go to the Statistics section, and click "Clear Cache". You can also view cached videos by clicking "View Cache".
Q: The extension skipped something it shouldn't have
A: You can increase the Confidence Threshold in Advanced Settings to make the AI more conservative. You can also whitelist specific channels.
Q: Is this open source?
A: Yes! The code is available at github.com/ChromuSx/SkipTubeAI
